## server

```bash
# init node project
npm init -y
# install packages
npm install sql.js
# just te be sure: ignore the node_modules folder
echo node_modules > .gitignore
```

```bash
# create files
mkdir public
mkdir public/js
mkdir public/css
mkdir src
touch public/css/style.css
touch public/js/app.js
touch public/index.html
touch src/index.js
```
